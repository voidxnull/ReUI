# Button
```js
import { Button } from 'reui';
```

### Example
```js
<Button title="Hello world!" />
<Button>Children here</Button>

// Active or "checked" button
<Button active title="Active button" />

// Disabled button
<Button disabled title="Disabled button" />
```

### Included themes
* ButtonPrimary
* ButtonSuccess
* ButtonInfo
* ButtonWarning
* ButtonDanger
